# Task Completion Checklist
- Run `pnpm lint` and `pnpm tsc` for code quality and type safety.
- Run `pnpm test` and relevant `pnpm test:e2e` / `pnpm test-storybook` when UI/behavior changes.
- Update `docs/` when specs or behavior change; add/update spec IDs for traceability.
- Ensure Node/Pnpm versions match `package.json` engines.
