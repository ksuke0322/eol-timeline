# Style & Conventions
- Language: TypeScript + React (React Router).
- Styling: Tailwind CSS (entry: `app/app.css`).
- Lint/format: ESLint flat config + Prettier integration (`eslint.config.mjs`).
- Imports: `import/order` enforced; alphabetical within groups.
- Naming: Components PascalCase, hooks `useXxx`, tests `*.test.ts(x)`, E2E `*.spec.ts`.
- Traceability: include spec IDs (e.g., `FR-SIDEBAR-001`) in `describe/it` names or comments when relevant.
