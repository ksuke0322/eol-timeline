# Suggested Commands
- `pnpm dev`: start dev server
- `pnpm build`: production build
- `pnpm start`: preview build output
- `pnpm typecheck`: React Router typegen + TS check
- `pnpm tsc`: type check only
- `pnpm lint`: ESLint with `--fix`
- `pnpm test`: Vitest run
- `pnpm test:ui`: Vitest UI
- `pnpm storybook`: Storybook dev
- `pnpm test-storybook`: Storybook tests
- `pnpm test:e2e`: Playwright E2E
- `pnpm all-check`: lint + tsc + test + build + storybook + e2e

System (Darwin): use `ls`, `rg`, `cd`, `git status`, `git log` as needed.
